## Regenerate preview screenshots

Preview images (`preview.png` in each template folder) can be regenerated with Playwright:

```bash
npm install
npx playwright install chromium
node capture-previews.mjs
```

## Customization

Every template can be fully customized:

- Colors, fonts, and layout
- Images, copy, and branding
- Sections added, removed, or reordered
- Contact forms wired to your email or WhatsApp

Typical delivery: **3–7 days** depending on scope and how quickly content is provided.
